# 卡片

## POST 申请开卡

POST /card/apply

用户申请新的虚拟卡或实体卡，需要先完成KYC认证并支付开卡费用

> Body 请求参数

```json
{
  "area_code": "+86",
  "name_on_card": "JOHN DOE",
  "phone": "13800138000",
  "physical": false,
  "postal_address": "某街道123号",
  "postal_city": "上海",
  "postal_code": "200000",
  "postal_country": "CN",
  "recipient": "张三"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|body|body|[handlers.ApplyCardSwaggerRequest](#schemahandlers.applycardswaggerrequest)| 是 |none|

> 返回示例

> 200 Response

```json
{
  "data": {
    "card_id": 100,
    "kyc_id": 12345,
    "physical": false,
    "task_id": "task_abc123"
  },
  "message": "Card application submitted",
  "status": "success"
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|申请成功|[handlers.ApplyCardSwaggerResponse](#schemahandlers.applycardswaggerresponse)|
|400|[Bad Request](https://tools.ietf.org/html/rfc7231#section-6.5.1)|KYC未通过或请求参数错误|[handlers.ErrorResponse](#schemahandlers.errorresponse)|
|401|[Unauthorized](https://tools.ietf.org/html/rfc7235#section-3.1)|未授权|[handlers.ErrorResponse](#schemahandlers.errorresponse)|
|402|[Payment Required](https://tools.ietf.org/html/rfc7231#section-6.5.2)|未支付开卡费用|[handlers.ErrorResponse](#schemahandlers.errorresponse)|
|500|[Internal Server Error](https://tools.ietf.org/html/rfc7231#section-6.6.1)|服务器内部错误|[handlers.ErrorResponse](#schemahandlers.errorresponse)|

### 属性

|名称|类型|必选|约束|中文名|说明|
|---|---|---|---|---|---|
|area_code|string|false|none||收件人电话区号（实体卡必填）|
|name_on_card|string|false|none||实体卡上印刷的名字（英文大写，实体卡必填）|
|phone|string|false|none||收件人电话（实体卡必填）|
|physical|boolean|false|none||是否为实体卡，false=虚拟卡，true=实体卡|
|postal_address|string|false|none||邮寄详细地址（实体卡必填）|
|postal_city|string|false|none||邮寄城市（实体卡必填）|
|postal_code|string|false|none||邮寄邮编（实体卡必填）|
|postal_country|string|false|none||邮寄国家/地区代码（实体卡必填）|
|recipient|string|false|none||收件人姓名（实体卡必填）|
